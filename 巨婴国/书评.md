# 书评：《巨婴国》

**作者：** 武志红
**出版时间：** 2016年12月
**类型：** 心理学/社会学

---

## 一、书籍简介

《巨婴国》是知名心理学家武志红创作的心理学著作，首次出版于2016年12月。这本书以独特的视角剖析了中国社会中普遍存在的"巨婴"现象——那些生理上已经成年，但心理上仍停留在婴儿阶段的人。

武志红用犀利而不失温情的笔触，揭示了"巨婴"现象背后的深层心理机制，以及这种现象对中国社会、家庭、亲密关系产生的深远影响。

---

## 二、核心概念

### 什么是"巨婴"？

"巨婴"是指生理成年但心理幼稚的人。他们具有以下典型特征：

| 特征 | 表现 |
|------|------|
| **全能自恋** | 认为自己是宇宙中心，世界应该围着自己转 |
| **偏执分裂** | 非黑即白的思维方式，难以接受事物的复杂性 |
| **共生心理** | 无法独立，必须与他人绑定才能感到安全 |
| **情绪失控** | 遇到挫折时像婴儿一样大发脾气 |
| **缺乏边界** | 不尊重他人的独立性，喜欢控制和干涉 |

### 中国式家庭的三重悲剧

武志红指出，巨婴心理在中国式家庭中形成了三重悲剧：

1. **巨婴父母养育巨婴孩子** — 父母自己没有长大，却要养育孩子
2. **共生绞杀** — 父母以"爱"的名义控制孩子，让孩子无法独立
3. **孝顺文化绑架** — 孩子被迫满足父母的心理需求，牺牲自我

---

## 三、精彩观点摘录

> "中国人的集体心理年龄没有超过1岁。"

这句争议性的论断，并非贬低，而是一种深刻的洞察。武志红认为，中国社会普遍存在"全能自恋"的心理状态——人们潜意识里认为世界应该满足自己的所有需求，一旦不被满足，就会产生强烈愤怒。

> "真正的爱，是让对方成为他自己。"

很多父母以为自己爱孩子，实际上是在爱"听话的孩子"。这种"爱"本质上是控制，剥夺了孩子成为独立个体的机会。

> "巨婴最怕的是孤独，因为孤独意味着要面对自己。"

巨婴无法独处，因为他们从未真正建立起独立的自我。他们需要不断寻找外部依赖——父母、伴侣、孩子、工作——来填补内心的空虚。

## 四、个人感悟

### 4.1 关于自我成长

读完这本书，最大的收获是对自我的觉察。我开始审视自己：

- 我是否在某些方面还像个"巨婴"？
- 我是否曾经用"爱"的名义控制他人？
- 我是否能够独立面对孤独？

成长的第一步，是承认自己可能还没有完全长大。这本书不是用来指责他人的，而是一面镜子，照见我们每个人内心深处那个还没有长大的部分。

### 4.2 关于亲密关系

《巨婴国》对理解亲密关系极有帮助。很多关系问题，本质上是一方或双方处于"巨婴"状态：

- 期望伴侣无条件满足自己
- 遇到冲突就情绪崩溃
- 无法接受伴侣的独立人格
- 用"你爱我吗"进行情感绑架

成熟的爱是什么？武志红给出的答案是：**允许对方成为他自己，同时你自己也成为你自己。**

### 4.3 关于代际创伤

书中最触动的部分，是关于代际创伤的描述：

> "如果你没有被好好爱过，你就很难好好爱别人。"

很多"巨婴"父母，自己就是被"巨婴"养育长大的。他们在童年时期没有得到真正的爱，所以也不知道如何真正地爱孩子。这是一种代代相传的创伤。

打破这个循环，需要我们首先意识到它的存在，然后有意识地去改变。

## 五、争议与批评

《巨婴国》并非没有争议。一些批评认为：

1. **以偏概全**：将中国社会的问题过度简化为"巨婴"心理
2. **文化偏见**：用西方心理学框架评判中国文化
3. **标签化风险**：容易让人对他人贴标签，而非真正理解

这些批评有其合理性。但作为一本引发思考的书，《巨婴国》的价值在于让我们开始审视那些习以为常的现象，而非给出标准答案。

## 六、谁适合读这本书？

| 适合人群 | 原因 |
|----------|------|
| 正在经历亲密关系困扰的人 | 理解关系中的心理动力 |
| 为人父母或即将为人父母 | 避免"巨婴式"养育 |
| 正在进行自我成长的人 | 获得深度的自我觉察 |
| 对中国式家庭关系困惑的人 | 理解家庭背后的心理机制 |
| 心理学爱好者 | 了解精神分析在中国的应用 |

## 八、总结

《巨婴国》是一本值得反复阅读的书。它不是一本让你读完就"好了"的书，而是一本需要你在人生不同阶段反复回来读的书。

每一次阅读，都会发现自己又有新的理解和感悟。因为我们每个人，都在不断成长的过程中，而这本书，正是成长路上的一盏灯。

**推荐指数：⭐⭐⭐⭐⭐**

如果你觉得自己在某些方面还没有完全长大，如果你对亲密关系感到困惑，如果你想理解中国式家庭的深层问题，这本书值得一读。
