# 书评：《中国传统文化的陷阱》

**作者：** 端木赐香
**出版社：** 长征出版社
**出版时间：** 2005年9月（第一版）
**类型：** 杂文/文化批判/思想评论

---

## 一、书籍简介

《中国传统文化的陷阱》是学者端木赐香的杂文集，以犀利的笔触剖析中国传统文化中的"陷阱"——那些看似美好、实则有害的文化因素。

端木赐香是一位历史学者和杂文作家，以尖锐的文化批判著称。她的风格与柏杨有相似之处——都是杂文式的批判，不留情面，直戳痛处。书名"陷阱"二字点明主题：**传统文化不全是精华，还有很多容易被忽视的陷阱。**

## 三、核心问题：什么是"陷阱"？

### 2.1 "陷阱"的含义

书中讨论的"陷阱"，指的是传统文化中那些：

| 陷阱 | 表现 |
|------|------|
| **隐蔽** | 不容易被发现 |
| **美丽** | 表面看似美好 |
| **有害** | 实际上有害无益 |
| **根深** | 深植于文化基因 |

这些"陷阱"不是明显的"糟粕"，而是披着"精华"外衣的有害因素。

### 2.2 常见的"陷阱"

**1. "和为贵"的陷阱**

> "和为贵"看似美好，但容易变成无原则的妥协。

陷阱在于：

- 以"和"的名义压制不同意见
- 以"和"的名义容忍错误
- 以"和"的名义放弃原则

**2. "孝"的陷阱**

> "孝"是美德，但容易变成压抑个性的工具。

陷阱在于：

- 以"孝"的名义要求绝对服从
- 以"孝"的名义剥夺个人选择
- 以"孝"的名义维持不合理关系

**3. "面子"的陷阱**

> "面子"是人际润滑剂，但容易变成虚伪的根源。

陷阱在于：

- 以"面子"的名义说谎
- 以"面子"的名义做违心之事
- 以"面子"的名义浪费资源

**4. "中庸"的陷阱**

> "中庸"是智慧，但容易变成没有立场。

陷阱在于：

- 以"中庸"的名义回避问题
- 以"中庸"的名义不辨是非
- 以"中庸"的名义和稀泥

**5. "集体主义"的陷阱**

> "集体主义"看似高尚，但容易变成压抑个人的借口。

陷阱在于：

- 以"集体"的名义否定个人
- 以"集体"的名义压制异议
- 以"集体"的名义逃避责任

## 三、"陷阱"的危害

### 3.1 阻碍现代化

这些"陷阱"阻碍中国现代化：

| 陷阱 | 阻碍 |
|------|------|
| "和为贵" | 阻碍法治，变成无原则妥协 |
| "孝" | 阻碍独立人格，变成依附关系 |
| "面子" | 阻碍真诚沟通，变成虚伪文化 |
| "中庸" | 阻碍改革创新，变成保守借口 |
| "集体主义" | 阻碍个人发展，变成压抑工具 |

### 3.2 扼杀创造力

传统"陷阱"扼杀创造力：

- 不鼓励质疑，如何创新？
- 不鼓励个性，如何独特？
- 不鼓励冒险，如何突破？

### 3.3 制造人格扭曲

传统"陷阱"制造人格扭曲：

| 表面 | 实际 |
|------|------|
| 温和 | 压抑真实情感 |
| 和谐 | 表面和气、内心冲突 |
| 孝顺 | 被动服从、缺乏独立 |
| 中庸 | 没有原则、随波逐流 |

## 四、如何识别"陷阱"？

### 4.1 辨析的标准

识别"陷阱"，需要辨析：

**1. 是原则还是借口？**

> "和为贵"是原则，还是无原则妥协的借口？

**2. 是美德还是压迫？**

> "孝"是美德，还是压抑个性的压迫？

**3. 是智慧还是逃避？**

> "中庸"是智慧，还是回避问题的逃避？

**4. 是保护还是控制？**

> "集体主义"是保护，还是控制个人的借口？

### 4.2 辨析的方法

辨析"陷阱"的方法：

| 方法 | 内容 |
|------|------|
| **追问** | 这个观念到底服务于谁？ |
| **反思** | 它的后果是什么？ |
| **比较** | 其他文化如何看待？ |
| **实践** | 实际效果如何？ |

---

## 五、"陷阱"的根源

### 5.1 历史根源

这些"陷阱"有深刻的历史根源：

**专制制度的需要：**

> 专制制度需要"顺民"，不需要"公民"。

- "和为贵"培养顺从
- "孝"培养服从
- "中庸"培养听话

**农耕社会的产物：**

> 农耕社会需要稳定，不需要变革。

- 稳定优于创新
- 服从优于独立
- 集体优于个人

### 5.2 文化心理的延续

这些"陷阱"通过文化心理延续：

- 家庭教育传承
- 学校教育强化
- 社会舆论维护

## 六、如何跳出"陷阱"？

### 6.1 觉察是第一步

> "认识到陷阱，是跳出陷阱的第一步。"

觉察意味着：

- 承认传统文化的两重性
- 看到美好背后的有害
- 不盲从传统观念

### 6.2 批判继承

> "对传统文化，要批判继承，不是全盘继承。"

批判继承意味着：

| 继承 | 扬弃 |
|------|------|
| 真正的美德 | 变质的观念 |
| 人生智慧 | 压迫的工具 |
| 文化认同 | 落后的思想 |

### 6.3 建立新观念

跳出"陷阱"，需要建立新观念：

| 旧观念 | 新观念 |
|--------|--------|
| 和为贵 | 有原则的和谐 |
| 孝顺 | 有边界的爱 |
| 面子 | 真诚 |
| 中庸 | 有立场 |
| 集体主义 | 个人与集体平衡 |

## 九、个人感悟

### 9.1 "好观念"的变质

读这本书最大的感悟是：**好的观念也可能变质。**

"孝"、"和"、"中庸"、"面子"、"集体主义"——这些观念本身没有问题，但在实践中容易变质：

- 美德变成压迫
- 智慧变成逃避
- 润滑剂变成虚伪

关键不是观念本身，而是如何理解、如何实践。

### 9.2 觉察的重要

这本书提醒我们：**觉察是改变的第一步。**

很多时候，我们被"陷阱"困住却不自知：

- 认为"和为贵"就是无原则妥协
- 认为"孝"就是绝对服从
- 认为"中庸"就是没有立场

觉察到这些是"陷阱"，才有可能跳出。

### 9.3 批判继承

这本书让我思考：**如何批判继承传统文化？**

既不是全盘继承——那会被"陷阱"困住
也不是全盘否定——那会失去文化根基

而是：

- 识别什么是精华，什么是陷阱
- 继承精华，扬弃陷阱
- 在继承中创新

## 十一、总结

《中国传统文化的陷阱》是一本"清醒"的书。

它清醒地告诉我们：**传统文化不全是精华，还有很多陷阱。**

这些陷阱：

- 披着美丽的外衣
- 深植于文化基因
- 不容易识别
- 有害无益

识别这些陷阱，是跳出陷阱的第一步。

批判继承传统文化，意味着：

- 保留真正的精华
- 扬弃变质的观念
- 在继承中创新

这本书的价值在于：

> **让我们用批判的眼光看待传统文化，而不是盲目崇拜或全盘否定。**
